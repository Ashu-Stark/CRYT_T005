Welcome to our secure E-Voting System. We are committed to providing a transparent, secure, and efficient voting platform that ensures the integrity of the democratic process.

Our Mission

Our mission is to revolutionize the voting process by leveraging advanced cryptographic technologies to ensure vote security, anonymity, and verifiability while maintaining accessibility for all voters.

Security Features

Our system employs state-of-the-art encryption methods:

  • AES Encryption: Votes are encrypted using Advanced Encryption Standard (AES) to ensure confidentiality.
  • RSA Encryption: AES keys are further protected using RSA public-key cryptography.
  • Voter Authentication: Each voter is authenticated using a unique Voter ID.
  • One Vote Per Voter: The system ensures that each voter can only cast one vote.

How It Works

When you cast your vote:

  1. Your vote is encrypted using AES encryption with a randomly generated key.
  2. The AES key is then encrypted using RSA public-key cryptography.
  3. Both the encrypted vote and encrypted key are stored securely.
  4. Only authorized administrators can decrypt and view the results.

Transparency & Trust

We believe in transparency and accountability. Our system maintains a complete audit trail while protecting voter privacy. All votes are securely stored and can only be decrypted by authorized personnel using the private key.
